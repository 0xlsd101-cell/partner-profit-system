# V1.0 GitHub 发布说明

## 1. 版本信息

- 版本号：v1.0.0
- 版本名称：V1.0 网页稳定版
- 发布形态：React + TypeScript + Vite 静态网页应用

## 2. 发布内容

- 合伙人管理。
- 资金批次管理。
- 月度收益结算。
- 年度分红汇总。
- 分红支付记录。
- 收益计算器。
- CSV / XLSX 导出。
- 完整 JSON 备份与导入恢复。
- 全中文可视化界面。
- GitHub Pages 自动部署配置。

## 3. 核心验证结果

- P0 = 0。
- P1 = 0。
- test 通过：90 个用例。
- lint 通过。
- build 通过。
- 真实数据隔离试运行通过。
- 验收库 locked 结算通过。
- 锁定前备份、锁定后备份和备份恢复演练通过。
- 页面抽检未发现 JSON 原文、英文技术字段或英文状态值。

## 4. 已知非阻塞问题

- Vite chunk 体积提示。
- `exceljs / uuid` 存在已知中等风险，无 high / critical。
- 移动端专项 QA 可在 V1.1 继续完善。
- 桌面离线版尚未开发。

## 5. 使用限制

- 数据保存在当前浏览器本地。
- 不自动跨设备同步。
- GitHub Pages 只托管静态页面。
- 换设备或换浏览器前，请先导出完整 JSON 备份。
- 不要上传真实备份文件、真实报表、`release-evidence/` 或 `data/trial/`。

## 6. GitHub Pages 设置

1. 进入 GitHub 仓库。
2. 打开 Settings -> Pages。
3. Build and deployment 的 Source 选择 GitHub Actions。
4. 打开 Actions 查看部署结果。
5. 部署成功后访问：
   - 普通仓库：`https://<用户名>.github.io/<仓库名>/`
   - 用户站点仓库：`https://<用户名>.github.io/`
