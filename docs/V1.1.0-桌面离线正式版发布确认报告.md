## 6. 卸载 / 重装观察

| 阶段 | 安装目录 | 可执行文件 | SQLite 文件 | 结论 |
| --- | --- | --- | --- | --- |
| 卸载前 | 存在 | 存在 | 存在 | 已先备份 SQLite 并导出 JSON |
| 卸载后 | 不存在 | 不存在 | 仍存在 | 卸载程序未删除用户数据 |
| 重装后 | 存在 | 存在 | 仍存在 | 重装后继续读取原 SQLite |

观察结论：

- 卸载会删除安装目录和可执行文件。
- 卸载不会删除 Roaming 应用数据目录下的 SQLite 数据库。
- 重装后会自动读取原 SQLite 数据。
- 真实使用中仍建议用户在卸载或重装前导出完整 JSON 备份，并保留 SQLite 文件副本。
