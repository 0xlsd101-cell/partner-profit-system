## 2. 修复内容

1. 修复月度结算页“负责人专项收益”展示错误。
   - 修复前：页面展示误用负责人理论收益字段。
   - 修复后：页面从月度收益明细汇总负责人专项收益。
   - 脱敏基准核对：负责人专项收益 `7,096.77`，负责人理论收益 `12,096.78`。

2. 回归验证桌面版 SQLite 清除本地数据。
   - 清除后业务表为空。
   - 保留 `app_metadata`。
   - 保留 `schemaVersion = 1`。
   - 关闭重开后空库状态保持。
