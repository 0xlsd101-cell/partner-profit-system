## 4. 修复说明

### 4.1 资金批次重复显示修复

读取层保留旧数据兼容能力：如果旧数据只有资金流水，没有资金批次，系统仍会自动从入金流水派生资金批次。

新增去重规则：

- 先按资金批次 ID 去重。
- 再按 `合伙人 + 金额 + 起息日期` 业务键去重。
- 如果导入备份已经有对应资金批次，则不再从同一笔入金流水派生重复批次。

该修复只影响读取展示和旧数据兼容，不改变收益计算公式，不改变 locked 月份结果。

### 4.2 运行环境诊断增强

系统设置页新增只读运行环境信息：

- 当前版本：V1.1.2
- 运行模式：桌面离线版 / 网页版
- 存储模式：本地 SQLite / 浏览器本地数据
- 本地数据库路径：桌面版显示 SQLite 文件位置

此项用于确认用户是否打开了新版本桌面程序，或仍在通过旧快捷方式启动 1.1.0。
