## 4. 人工验收建议

1. 使用脱敏 JSON 备份，不使用真实合伙人姓名和敏感金额。
2. 在桌面窗口进入“数据备份”页面。
3. 点击“选择备份文件”，选择脱敏 JSON。
4. 点击“查看导入摘要”，确认摘要为中文。
5. 点击“确认导入”，核对月度结算和年度汇总金额。
6. 导出 JSON / CSV / XLSX 文件，确认文件可打开且无英文技术字段或 JSON 原文暴露。
