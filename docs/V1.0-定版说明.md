# V1.0 定版说明

## 1. 版本名称

合伙人收益系统 V1.0 网页稳定版

## 2. 定版日期

2026-06-30

## 3. 版本定位

- 本地网页版稳定版。
- 支持合伙人资金管理。
- 支持月度收益结算。
- 支持年度分红汇总。
- 支持实际收入对账。
- 支持 CSV / XLSX 导出。
- 支持完整备份与恢复。
- 暂不是 Tauri + SQLite 桌面离线版。

## 4. 核心业务规则

- 普通合伙人使用年化单利收益率。
- 折合月收益率 = 年化收益率 ÷ 12。
- 按自然月结算。
- 月中加入首月按实际计息天数折算。
- 第二个月开始按整月计息。
- 负责人自有资金按本月总收益率计算。
- 负责人专项收益基于非负责人折算本金。
- 外部资金差额留存基于非负责人折算本金。
- 尾差默认归负责人。
- 年度周期为公历自然年度：每年 1 月 1 日至 12 月 31 日。
- locked / adjusted 月份不可直接修改。
- 历史修正必须通过调整记录处理。

## 8. 使用注意事项

- 录入真实月份前先导出完整备份。
- 锁定前必须人工复核金额、收益率、实际收入和备注。
- 锁定后不要直接修改历史月份，应使用调整记录处理修正。
- JSON 备份文件不要手动编辑。
- `release-evidence/v1.0/` 可能包含验收数据，不建议外发。
- 对外发布包应使用空库版本，不夹带真实业务数据、试运行数据或验收库备份。
