## 5. 卸载 / 重装观察

| 阶段 | 安装目录 | 可执行文件 | SQLite 文件 | 说明 |
| --- | --- | --- | --- | --- |
| 卸载前 | 存在 | 存在 | 存在，319,488 字节 | 已先备份 SQLite 和导出 JSON |
| 卸载后 | 不存在 | 不存在 | 仍存在，319,488 字节 | 卸载程序未删除用户数据 |
| 重装后 | 存在 | 存在 | 仍存在，319,488 字节 | 重装后继续读取原 SQLite |

观察结论：

- 卸载桌面应用会删除安装目录和可执行文件。
- 卸载不会删除 `C:/Users/1566/AppData/Roaming/com.partnerprofit.system/partner-profit-system.sqlite`。
- 重装后应用会继续读取原 SQLite 数据。
- 真实使用时，卸载前仍建议先导出完整 JSON 备份，并保留 SQLite 文件副本。
