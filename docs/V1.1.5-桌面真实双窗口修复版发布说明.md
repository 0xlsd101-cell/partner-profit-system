- 版本号：v1.1.5
- 发布定位：桌面版真实启动双窗口修复版
- 推荐状态：当前推荐桌面稳定版

## 2. 修复内容

1. 修复 Windows release exe 启动时暴露 `ConsoleWindowClass` 黑色空白控制台窗口的问题。
2. 保留 `single-instance` 保护，重复启动时聚焦已有窗口，不新建第二个主窗口。
3. 修正窗口标题为真实版本：`合伙人收益系统 V1.1.5`。
4. 保持桌面快捷方式和开始菜单快捷方式指向新版安装目录。
