## 最小补齐路径

1. 用真实数据替换 `data/trial/*.csv`。
2. 重新执行真实数据隔离试运行并生成新报告。
3. 录入 1 个真实月份草稿，不自动锁定。
4. 导出 CSV / XLSX 给负责人和相关合伙人人工复核。
5. 导出锁定前完整备份。
6. 人工确认后再锁定试运行月份。
7. 导出锁定后完整备份。
8. 在测试环境完成备份恢复演练并记录结果。
9. 再次执行 `test`、`lint`、`build`、`npm audit` 后进入 V1.0 最终判定。
