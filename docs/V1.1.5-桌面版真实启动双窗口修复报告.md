# V1.1.5 桌面版真实启动双窗口修复报告

## 1. 问题结论

- 是否复现双窗口：已基于用户真实桌面快捷方式复现。
- 是否两个进程：否。修复前只有一个 `partner-profit-system.exe` 进程。
- 是否同一路径：是。进程路径为 `C:\Users\1566\AppData\Local\Programs\PartnerProfitSystemV11Test\partner-profit-system.exe`。
- 根因：Windows release 可执行文件仍带控制台子系统，真实桌面快捷方式启动后，同一进程除 Tauri 主窗口外，还暴露了一个 `ConsoleWindowClass` 控制台窗口，标题为“合伙人收益系统”，表现为黑色空白窗口。
- 是否修复：已修复。
- 是否建议发布 v1.1.5：建议发布。
