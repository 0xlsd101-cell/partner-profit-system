# V1.1.3 桌面稳定修复版发布说明

## 1. 版本信息

- 版本号：v1.1.3
- 发布定位：桌面版稳定性修复版
- 适用范围：V1.1.x Tauri + SQLite 桌面离线版
- Web 版影响：不修改 Web 版收益业务逻辑，不影响 GitHub Pages 网页版。

## 2. 修复内容

1. 修复旧版桌面 / 开始菜单快捷方式可能仍指向 `1.1.0` exe 的问题。
2. 修复导入备份后，资金批次与资金流水派生批次重复显示的风险。
3. 增加系统设置页运行环境只读信息：
   - 当前版本
   - 运行模式
   - 存储模式
   - SQLite 数据库路径
4. 优化清除本地数据反馈：
   - 显示“正在清除本地数据...”
   - 清除完成后显示“本地数据已清除。”
5. 完成桌面版 GUI 导入、导出、恢复、清除本地数据、关闭重开全链路验证。
